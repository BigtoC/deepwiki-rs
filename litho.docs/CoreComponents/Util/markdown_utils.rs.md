`markdown_utils.rs` 是一个专门用于处理和生成Markdown格式文本的工具组件。它提供了丰富的功能来创建各种Markdown元素，包括标题、代码块、表格、列表、图片、链接等，以及处理Markdown特殊字符的转义和转换。该组件旨在简化Markdown内容的生成和管理，提高开发效率。
